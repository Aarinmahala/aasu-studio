AaSU STUDIO WEBSITE
====================

IMPORTANT: If you're having trouble viewing the website, please follow these instructions:

OPTION 1: VIEW DIRECTLY (EASIEST)
---------------------------------
1. Double-click on "index.html" to open it directly in your browser
   - This will show you the complete website with all features
   - No server required

OPTION 2: START THE SERVER (IF NODE.JS IS INSTALLED)
---------------------------------------------------
1. Double-click on "run-server.bat"
2. A command prompt window will open
3. If successful, you'll see a message saying the server is running
4. Open your browser and go to: http://localhost:8080
5. For the demo page, go to: http://localhost:8080/demo.html

TROUBLESHOOTING
--------------
If you see "localhost refused to connect" errors:
1. Make sure no other program is using port 8080
2. Try using a different browser
3. Disable any firewalls temporarily
4. Use Option 1 instead (direct viewing)
