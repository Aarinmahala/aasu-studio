## AaSU STUDIO

A professional photography and videography studio website showcasing creative work, services, and booking options.

![AaSU STUDIO](https://github.com/Aarinmahala/aasu-studio/blob/main/aasu-studio-website/public/images/logo.png)

## 📷 About

AaSU STUDIO specializes in high-quality photography and videography services for events, portraits, commercial projects, and more. The website provides a modern, responsive platform to showcase the studio's portfolio and services.

## ✨ Features

- Responsive design optimized for all devices
- Interactive portfolio gallery
- Service booking system
- Contact form with email integration
- Modern animations and transitions
- Dark mode theme with striking visuals

## 🚀 Live Demo

Visit the live website: [AaSU STUDIO](https://aarinmahala.github.io/aasu-studio)

## 🛠️ Technologies Used

- HTML5, CSS3, JavaScript
- React.js framework
- Node.js for server functions
- EmailJS for contact form
- AOS (Animate On Scroll) library
- Responsive design with media queries

## 🔧 Installation & Setup

1. Clone the repository:
   ```
   git clone https://github.com/Aarinmahala/aasu-studio.git
   ```

2. Navigate to the project directory:
   ```
   cd aasu-studio/aasu-studio-website
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```
   
   Or use the included batch file (Windows):
   ```
   start-server.bat
   ```

5. Open your browser and go to `http://localhost:3000`
